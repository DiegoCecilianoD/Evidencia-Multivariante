# Evidencia-Multivariante
Este proyecto realiza un análisis discriminante lineal (LDA) para estudiar los determinantes de la vulnerabilidad social en los estados de México en los años 2016 y 2022. El objetivo es identificar qué indicadores socioeconómicos permiten separar los estados según su nivel de vulnerabilidad y cómo estos factores cambiaron a lo largo del tiempo.
